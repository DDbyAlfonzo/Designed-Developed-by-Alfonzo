# DDbyAlfonzo

## Scripts

- `npm run dev` starts a local Vite server on `http://127.0.0.1:4173`
- `npm run preview` serves the production preview on the same port
- `npm run format` formats the site, tests, and config files with Prettier
- `npm run lint` runs HTML, CSS, and JavaScript checks
- `npm run test:visual` runs the Playwright homepage smoke tests
- `npm test` runs linting and the visual smoke tests together
